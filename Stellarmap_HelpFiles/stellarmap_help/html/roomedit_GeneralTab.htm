Room Editor: General Tab

This tab consists of several basic properties that should be set for all rooms. Most of these properties are presentational in manner and have little mechanical value in terms of gameplay.

Room Name

SetShort(string)

The title of the room that appears at the top of the description when the player enters. Can take a string or a value escaped with $.


Light

SetLight(int) or SetProperty("light",int)

Represents the brightness of a room. Default value is 30. For stellarmass anything below 20 is considered low-light and anything below 12 is near pitch black. As well, anything over 50 is over-bright.


Climate

SetClimate(string)

Sets the weather patterns for the room. Items are loaded from the configuration file.


Long Description

SetLong(string)

The detailed description of the room. Viewed when the player first enters the room (if player has long descriptions turn on) or when using the look command.


Enable Day/Night

n/a

This is just a switch to enable or disable the text boxes for Day and Night long descriptions. It does not generate any code for itself but instead allows SetDayLong() and SetNightLong() to be output to the file when saved.


Day Description

SetDayLong(string)

An alternate to SetLong() that is to be used in tandem with SetNightLong(). It sets the long description to be displayed during the daytime. Only works for rooms with a climate set to something other than indoors.


Night Description

SetNightLong(string)

An alternate to SetLong() that is to be used in tandem with SetDayLong(). It sets the long descriptio to be displayed during the night time. Only works for rooms with a climate set to something other than indoors.


Icon

n/a

Chooses the icon to display on the room in the map domain view. This does have any effect on the code generated for this room.
