Room Editor: Properties Tab

Various gameplay elements can be changed in this tab. It included features that affect what players can carry due to weight, what special actions they can or cannot perform, and how they can move about.

Gravity

SetGravity(int)

Affects the weight of the objects the player is carrying.


Town

SetTown(string)

Tags a room as belonging to a specific township. Used when allowing players to create their own homes within a region. Laregly unused in Stellarmass at this time. See http://dead-souls.net/docs/build/Towns for more details.


Obvious Exits

SetObviousExits(string)

Text that is displayed at the bottom of the room description for avilable exits. Useful for making certain exits 'hidden'. For example if a room has "avilable exits: north, east, south" by default, you could enter "north, south" and thus make' east' invisible.


Leave Tracks

SetTracksAllowed(int)

If checked, any living creature that passes through this room will leave tracks for a short time. Recommended to be left on so that Explorers can use their 'Track' skill.


Room Properties

SetProperties(mapping) or SetProperty(string,mixed)

Controls certain aspects of a room.

  • no attack - Disallows players and NPCs from attacking anything
  • no steal - Disallows players and NPCs from stealing from anyone
  • no magic - Disables powers. Currently the 'powers' system is used by Stellarmass however, if it ever is, this option will disable class-specific skills.
  • no bump - Keeps players and NPCs from forcing others out of the room and into adjacent rooms
  • no paralyze - Keeps players and NPCs from becoming helplessly paralyzed, even by lack of stamina
  • no teleport - Disables teleporting into or from the room(and maybe
  • no clear - prevents an avatar from clearing a wilderness room in order to build a town. Not relevant to rooms in towns.
  • login - Flags this room as the place the player will return when they log out. Note, this is Stellarmass specific and slightly different from Standard Dead souls.
  • hospital - Same thing as login but also flags the room a player will return to if they die or if they use the paramedic command. You should always place a doctor in the room if you use this so that players can regenerate health and limbs.
  • mechs allowed - Allows players driving mechs to enter this room.


Terrain Types

SetTerrain(int)

Sets the atmosphere, legal movement types, and stamina use required by players moving through a room. Some combinations are illegal (for example, underwater and road). Beware, some of these choices are leathal to exposed players; such as T_SPACE. Some are meant to be used as combinations with the room's Medium (such as T_SURFACE and MEDIUM_WATER).


Medium

SetMedium(int)

Another type of specification for atmoshpere and sometimes movement capacity. Note, MEDIUM_VACCUM is a costume value for stellarmass and represents a region with solid ground and no atmosphere as opposed to MEDIUM_SPACE which has neither ground nor atmsphere.
