Item Generator: Weapon Stats

All items have the default ability to be used as a weapon. The idea is that you could technically stab someone with a key or bludgeon them with a book if you needed to in a pinch. Here is where you can define such damage properties. Keep in mind, this does not affect firearms' shooting properties. For that you need to create a firearm item and go to the Inherited Properties tab. However, this does allow you to set the damage properties of a firearm's melee attack mode.

Damage Types

SetDamageType(int)

The total combine types of damage dealt by this weapon. You can select multiple damage types to combine effects. For example, a flaming sword might be HEAT and BLADE. An explosive shotgun round might be BALLISTIC, HEAT, and TRUAMA. Carrot Top's comedy routine might deal EMOTIONAL and TRUAMA damage. In some cases armor may only protect against certain typres of damage and if it doesn't cover a category declared for the weapon then some of that damage 'slips through' unchanged.


Custom Properties

SetProperties(mapping)

Simply applies anyadditional properties. Some properties may simply be for storing additional data for specific in-game use. There are, however, a few pre-defined properties for items in DS. Check this page for properties pretaining to items. http://dead-souls.net/docs/build/Properties


Max Damage

SetClass(int)

The maximum damage that a weapon might deal before being adjusted by character stats and randomness. The final in-game value may be higher or lower depending on skills, stats, random numbers, and other combat situations.


Hands Required

SetHands(int)

The number of free hands needed to wield this item as a weapon


Value Half-Life

SetDamagePoints(int)

The number of points of damage this item can take before it's value decreases by half. When they reach zero they are reset.


Wield Message

SetWield(string)

The message displayed to the user when they wield this item as a weapon. If empty the default wield message will be displayed.


Wield Level

SetWieldLevel(int) Stellarmass-only Feature

The minimum level a character must be in order to wield this item as a weapon.
