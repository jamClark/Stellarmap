Item Generator: Ammo Inherited Features

Here you will find the properties of an item that represents ammo to be used with firearms. The most important property is the Ammo Type which identifies what types of weapons it can be used in. Everything else is optional.

Ammo Type

SetAmmoClass(string)

A string of text that identifies what types of weapons can load this ammo. The Ammo Type of the firearms meant to load it should be the same value.


Max Ammo

SetMaxAmmo(int)

For magazine based ammo, this number will be how many round are stored inside of it. Note: If this is meant to be a single round, for example, a shot gun shell or an arrow, then this value must stay at 1. The firearm itself will keep track of the ammo count in such cases and certain assumptions are made about single-shot ammo. Also take note that any round that has a max ammo count of 1 will be treated as a round regardless of the type of weapon it is in.


Ammo Damage Type

SetDamageTypeBonus(int)

This is a bonus damage type that the ammo adds to the default damage type already done by the weapon.


Damage Bonus

SetClassBonus(int)

Bonus damage dealt by the ammo that is added to the firearm's max damage before adjustsments of stats, skills, randomness, etc.


Accuracy Bonus

SetAccuracyBonus(int)

Additional accuracy applied to the weapon when firing this type of ammo. Note that the final resulting accuracy will never be over 100%.
