Room Editor: Room Details Tab

These are extra details that can be added to the room. Usually this is where static items are given the additional descriptions for searching, listening to, smelling, and reading. The term 'default' is defined for each as a case that happens when a player issues a command without speciying what. For ecample if player types search but not search desk then you can issue a message for a default search of the room or something similar.

Searchables

SetSearches(mapping)

The result of the player issuing the searchcommand. The right box should either contain text for the result to display or an $-escaped function/functional that performs additional processing.


Readables

SetReads(mapping)

The result of the player issuing the readcommand. The right box should either contain text for the result to display or an $-escaped function/functional that performs additional processing.
